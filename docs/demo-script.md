# AgentOps 360 — Demo Script

## Demo Goal

Show how AgentOps 360 turns messy enterprise meetings into clear action plans.

---

## Demo Flow

### 1. Open the dashboard

Explain:

> AgentOps 360 is an autonomous enterprise operations agent. It transforms meetings, documents, and audio transcripts into decisions, tasks, risks, follow-up emails, and executive reports.

---

### 2. Select or create a project

Explain:

> Each analysis belongs to a project workspace. This allows teams to keep an auditable history of decisions and execution plans.

---

### 3. Paste or upload meeting notes

Use this scenario:

```txt
The sales team discussed a drop in conversion rate after the new pricing page launch. Laura will analyze the funnel data tomorrow. The design team must prepare two new landing page variations. The CEO wants a recovery plan before Friday.
```

Explain:

> This is an unstructured business note. The goal is to convert it into a structured execution plan.

---

### 4. Click Run Autonomous Analysis

Explain while the pipeline runs:

> The workflow is split into specialized agents: intake, reasoning, planning, communication, and audit.

---

### 5. Show the result

Highlight:

- Confidence score
- AI Mode: Gemini Live
- Decisions
- Action plan
- Risk matrix
- Follow-up email
- Agent trace

---

### 6. Show analysis history

Explain:

> Every run is stored in SQL Server and can be reopened later. This creates an execution memory for the organization.

---

### 7. Export the report

Click Export Report.

Explain:

> Managers can immediately share the executive report after the meeting.

---

## Closing Pitch

AgentOps 360 reduces the gap between meetings and execution.

It helps enterprises:

- Save time after meetings
- Clarify decisions
- Assign owners
- Detect risks early
- Generate follow-up communication
- Keep an auditable history

This turns meetings from passive conversations into autonomous execution workflows.
